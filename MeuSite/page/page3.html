Exercício 2: Formatação de Texto e Blocos Préformatados

Formatação de Texto

b e strong

Usar negrito para deixar o texto mais destacado visualmente, mas sem indicar importância semântica. Já forte indica que o texto é realmente importante, e os leitores de tela reconhecem isso.

i e em

O itálico serve apenas para mudar o estilo do texto, enquanto o ênfase indica que a palavra ou frase deve ser enfatizada, ajudando na interpretação semântica.

mark

Use marcações para destacar uma palavra ou trecho específico que precisa chamar atenção.

small

Um texto com tamanho menor pode ser usado para observações, avisos ou informações secundárias.

del e ins

Aqui temos um texto removido e outro inserido para mostrar alterações ou correções.

sub

Um exemplo de subscrito é a fórmula da água: H2O.

sup

Um exemplo de sobrescrito é para potências: X2.

Bloco de Texto Pré-formatado

A tag "pre" é usada para mostrar texto preservando os espaços em branco e as quebras de linha exatamente como estão no código-fonte. Isso é útil para exibir trechos de código ou desenhos feitos com caracteres, chamados de "arte ASCII", sem que o navegador reorganize ou quebre o texto automaticamente. 

# Exemplo de código Python
for i in range(3):
    print("Olá, mundo!")
